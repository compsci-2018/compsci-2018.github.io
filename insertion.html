Insertion Sort!

The 2nd element in a list, (since the 1st element is already sorted)



acts as a bridge between two parts, a sorted list and an unsorted list.
Our goal is to move all the unsorted elements over the bridge and sort them out on the other side

Quick Facts

How it works:

find the smallest element, and swap it with a[0]... and so on: find the smallest element and swap it with a[n]

For an array of n...

the array is sorted after n-1 passes

After the kth pass..

the first k elements are in their final sorted position



Ready for review?

Starting with the 2nd element at array[1].

Given these elements

43 32 76 99 18 98

The smallest element is 2! So now we exchange this with a[0]

2 7 9 4 Since we already exchanged an element at a[0], we move on to the next element at a[1].

Again, we look for the smallest element.
which is 4

Since we already exchanged an element at a[0], we move on to the next element at a[1].
Exchange 4 with 7

2 4 9 7

Getting the hang of it? 7 is our smallest element now. We already swapped with the element at a[0] and a[1]
so now lets exchange 9 and 7

2479
We're done! All elements are in order.
